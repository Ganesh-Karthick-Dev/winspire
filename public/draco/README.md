# Draco WASM Decoder Files

Place the Draco decoder files here. Download from:
https://github.com/mrdoob/three.js/tree/dev/examples/jsm/libs/draco

Required files:
- draco_decoder.wasm
- draco_decoder.js
- draco_wasm_wrapper.js

These files are required for loading Draco-compressed GLTF/GLB models.
