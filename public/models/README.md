# GLTF/GLB Models

Place your optimized 3D models here.

Expected file: hero-draco.glb

Use the optimization script to process your models:
```bash
npm run optimize-model -- input.glb public/models/hero-draco.glb
```

Or manually:
```bash
bash scripts/optimize-model.sh input.glb public/models/hero-draco.glb
```
