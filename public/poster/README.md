# Poster Images

Place your hero poster image here for LCP optimization.

Expected file: hero-poster.webp

This image is:
- Preloaded for fast LCP
- Shown while 3D model loads
- Displayed as fallback on low-end devices
- The actual LCP element (not the 3D canvas)

Recommended specs:
- Format: WebP
- Size: 1920x1080 or similar 16:9 ratio
- Quality: 80-90
- File size: < 200KB
